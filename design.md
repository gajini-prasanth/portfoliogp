# Portfolio Design Style Guide

## Design Philosophy

### Color Palette
- **Primary**: Deep Teal (#0D4F4C) - Professional, trustworthy, tech-focused
- **Secondary**: Warm Coral (#FF6B6B) - Energetic, creative, approachable  
- **Accent**: Soft Gold (#F7B731) - Achievement, innovation, premium feel
- **Neutral**: Charcoal Gray (#2C3E50) - Modern, sophisticated, readable
- **Background**: Off-White (#FAFBFC) - Clean, spacious, professional

### Typography
- **Display Font**: "Playfair Display" - Elegant serif for headings and hero text
- **Body Font**: "Inter" - Clean, modern sans-serif for readability
- **Code Font**: "JetBrains Mono" - Technical, precise for code snippets

### Visual Language
- **Minimalist Sophistication**: Clean lines, generous whitespace, purposeful elements
- **Tech-Forward Aesthetic**: Subtle tech patterns, geometric shapes, modern UI elements
- **Personal Touch**: Warm colors balanced with professional presentation
- **Interactive Elegance**: Smooth animations that enhance rather than distract

## Visual Effects & Styling

### Used Libraries
- **Anime.js**: Smooth page transitions, element animations, timeline interactions
- **ECharts.js**: Interactive skills radar chart with hover effects
- **p5.js**: Subtle particle background effects and creative coding elements
- **Splide.js**: Smooth project carousel and image galleries
- **Pixi.js**: Advanced visual effects for hero section

### Animation & Effects
- **Hero Section**: Animated particle system with floating code elements
- **Skills Visualization**: Interactive radar chart with smooth data transitions
- **Project Timeline**: Horizontal scroll with parallax effects and smooth reveals
- **Navigation**: Subtle hover animations with color transitions
- **Scroll Effects**: Gentle parallax on background elements, staggered content reveals

### Header Effect
- **Gradient Flow Background**: Subtle animated gradient using CSS and p5.js
- **Floating Elements**: Geometric shapes and code snippets with physics simulation
- **Typography Animation**: Staggered letter reveals for name and title
- **Interactive Particles**: Mouse-responsive particle system

### Styling Approach
- **Card-Based Layout**: Modern card design with subtle shadows and hover effects
- **Responsive Grid**: Flexible layouts that work across all devices
- **Micro-Interactions**: Button hover states, form field animations, loading states
- **Consistent Spacing**: 8px grid system for perfect alignment
- **Accessibility**: High contrast ratios, keyboard navigation, screen reader support

### Visual Hierarchy
- **Hero Impact**: Large, bold typography with animated background
- **Content Sections**: Clear separation with consistent spacing and typography scales
- **Interactive Elements**: Subtle visual cues for clickable items
- **Data Visualization**: Clean, readable charts with consistent color usage
- **Call-to-Actions**: Prominent buttons with hover animations and clear messaging
